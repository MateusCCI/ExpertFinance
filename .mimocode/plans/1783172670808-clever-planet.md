# Plano: Cartoes virtuais no formulario de novo cartao

## Contexto
Cartao virtual = mesmo cartao fisico com ultimos 4 digitos diferentes (Apple Pay etc). Precisa ser adicionado junto com o fisico no mesmo form.

## Arquivo: src/pages/CreditCards.tsx

## 4 mudancas

### 1. NewCardForm - substituir state isVirtual/parentCardId
Remover `isVirtual` e `parentCardId`. Adicionar:
```
virtuals: Array<{ lastDigits: string; name: string }> 
```
Estado inicial: `[]` (sem virtuais)

### 2. NewCardForm - adicionar UI apos "Meta de gasto"
- Botao "Adicionar cartao virtual" que adiciona item ao array `virtuals`
- Cada virtual: input "Nome" (ex: Apple Pay) + input "Ultimos 4 digitos" + botao X remover
- Quando virtuals.length > 0, o form ja cria o fisico e os virtuais juntos

### 3. onSave callback - trocar parentCardId por virtuals
Tipo muda de `parentCardId: string | null` para:
```
virtuals?: Array<{ name: string; lastDigits: string }>
```

### 4. handleSaveCard - criar virtuais apos fisico
- Criar cartao fisico normalmente (parent_card_id: null)
- Se data.virtuals.length > 0, para cada virtual:
  - createCard com parent_card_id: newCard.id
  - Copiar brand, limit, dates, color do fisico
  - Usar nome e lastDigits do virtual
- Dialog novo cartao: passar physicalCards prop
- Dialog editar: passar physicalCards prop

## Verificacao
- Abrir form novo cartao, preencher dados basicos
- Clicar "Adicionar cartao virtual", preencher dígitos
- Salvar -> deve criar 1 fisico + N virtuais vinculados
- Na tela de cartoes, virtuais aparecem aninhados no fisico
